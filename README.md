# fuzzy-meme

Made using [create-react-app](https://github.com/facebookincubator/create-react-app)

## Getting Started

`$ npm start`

# Start server only
`$ npm run server`

# Run documentation
`$ apidoc -e node_modules`

# Reinitialize database (from root)
`$ npx babel-node server/database/reinitialize`
